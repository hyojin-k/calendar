# calendar_project

## 캘린더 만들기
- 리액트로 일정 관리 캘린더 만들기

### 기술 스택
- React
- Redux
- styled-components
- firebase
- fullcalendar
- react-bootstrap
  


### 주요 기능
- 메인 페이지

    - 달력 뷰 구현하기 

    - 일정 클릭시 모달 페이지로 상세 일정 보여주기

    - 전체 일정 / 완료 일정 토글 사용하여 보여주기

- 상세 페이지

    - 모달 페이지에서 삭제, 완료 기능 구현
    
    - 해당 일정 완료 시 색상 변경하기  

- 일정 등록 페이지

    - 일정, 시간 입력하여 일정 추가하기
